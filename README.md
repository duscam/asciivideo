# asciivideo

## Install requirements

### Create virtualenv
python -m virtualenv venv

### Activate env
#### MacOS/Linux:
. venv/bin/activate

#### Windows:
.\venv\Scripts\activate

### Install requirements
pip install -r requirements.txt

## Run app
Converts a .mp4 video into an ascii representation of the frames

  `python video_to_ascii.py`
